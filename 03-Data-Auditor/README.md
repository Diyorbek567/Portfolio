\# 🛠 AI Data Auditor Pro



Katta hajmdagi Excel va SQL ma'lumotlarini avtomatik audit qiluvchi va tozalovchi interaktiv vosita.



\### 🌟 Asosiy xususiyatlar:

\- \*\*Fuzzy Matching:\*\* Ma'lumotlar ichidagi dublikatlarni va o'xshash xatoliklarni topish.

\- \*\*Excel Analysis:\*\* Bir nechta faylni solishtirish va farqlarni hisobot qilish.

\- \*\*Interactive UI:\*\* Streamlit yordamida foydalanuvchi uchun qulay interfeys.



\### 🛠 Texnologiyalar:

\- Python (Streamlit, Pandas, NumPy,thefuzz), sqlite3.

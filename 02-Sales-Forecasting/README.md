\# Retail Sales Forecasting Model



Ushbu loyiha savdo hajmini o'tmishdagi ma'lumotlar (historical data) asosida bashorat qilish uchun Machine Learning modelini qo'llaydi.



\### 🌟 Asosiy xususiyatlar:

\- \*\*Data Cleaning:\*\* Sotuvlar tarixini tahlil qilish va trendlarni aniqlash.

\- \*\*Predictive Modeling:\*\* Kelgusi davr uchun zaxiralarni optimallashtirish maqsadida bashorat qilish.

\- \*\*Visuals:\*\* Sotuvlar dinamikasini vizuallashtirish.



\### 🛠 Texnologiyalar:

\- Python, Scikit-learn, RandomForestRegressor,RandomForestClassifier, Pandas, Matplotlib,seaborn.
